Hex Hunter

Computers store every colour as three amounts of light — Red, Green and Blue — written as a hex code like #33CC99. Learn to read the code and hunt down the matching colour!

A hex colour is #RRGGBB. Each pair goes from 00 (none) to FF (255 = maximum).
#FF0000 red   #00FF00 green   #0000FF blue
#FFFFFF white (all on)   #000000 black (all off)
